Travel Document Verifier

The Travel Document Verifier was started in the summer of 2019. It is coded with C++ and it checks various travel document types according to checksum standards set by the ICAO. Currently, it can read identification strings in passports, Travel Document Types 1 and 2, and Visa Types A and B. You can read the full README at the GitHub link which the button provides.
